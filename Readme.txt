Καταρχάς απαιτείται η εγκατάσταση Mariadb, Mysqlworkbench (ή παρόμοιου εργαλείου) και python.


Για τη δημιουργία της βάσης πρέπει να εκτελεστούν τα αρχεία Schema.sql και Dummy Data.sql στον φάκελο Sql code, με αυτή τη σειρά.


Ο χρήστης πρέπει επιπλέον να εκτελέσει pip install -r requirements.txt από το terminal, όντας φυσικά στο σωστό directory.


Τέλος, εκτελούμε το run.py, το οποίο δημιουργεί τη σελίδα στο localhost, στην πόρτα 3000. Αρκεί λοιπόν να μεταφερθούμε στο localhost:3000 στο browser της επιλογής μας.
